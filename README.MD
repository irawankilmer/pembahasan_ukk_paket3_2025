# Pembahasan Soal Paket 3 UKK Tahun Ajaran 2024/2025
## REKAYASA PERANGKAT LUNAK

Pada repo ini akan dibahas semua materi mengenai Uji Kompetensi Keahlian RPL Soal Paket 3, termasuk:
1. Dasar-Dasar Basis Data <br>
  - Pengertian dan konsep dasar basis data
  - Sistem Manajemen Basis Data (DBMS)
  - Jenis-jenis basis data
2. Perancangan Basis Data <br>
  - Analisis kebutuhan sistem
  - Diagram Entitas-Relasi (ERD)
  - Normalisasi basis data
3. Implementasi Basis Data <br>
  - Pembuatan tabel dan atribut
  - Relasi antar tabel (Primary Key & Foreign Key)
4. Bahasa SQL (Structured Query Language) <br>
  - DDL (Data Definition Language): CREATE, ALTER, DROP
  - DML (Data Manipulation Language): INSERT, UPDATE, DELETE, SELECT
  - DCL (Data Control Language): GRANT, REVOKE
5. Manajemen Data dan Query <br>
  - Pencarian data menggunakan SELECT dengan berbagai klausa (WHERE, ORDER BY, GROUP BY)
  - Fungsi agregat (COUNT, SUM, AVG, MIN, MAX)
  - Join dan subquery
  - Penggunaan indeks untuk optimasi pencarian data
6. Pengelolaan Transaksi dalam Basis Data <br>
  - Commit dan rollback
  - Penggunaan stored procedure, function, dan trigger
7. Backup dan Keamanan Basis Data <br>
  - Strategi backup dan restore data
  - Proteksi fisik dan lingkungan terhadap data
  - Enkripsi dan perlindungan informasi
  - Pengujian media backup dan prosedur restorasi
8. Pengelolaan dan Pemeliharaan Sistem Basis Data <br>
  - Monitoring performa basis data
  - Retensi dan pemulihan data
  - Dokumentasi sistem basis data
